# CloudFlare 解决 CORS 跨域问题

摘录：[CloudFlare 解决 CORS 跨域问题 - ROYWANG](https://roy.wang/cloudflare-cors-error/)

在域名-规则-转换规则处，新建**修改响应头**规则，按照下方图片，**值设置为自己的域名**

![1](https://pic.imgdb.cn/item/66fd46edf21886ccc02306e1.png)

下方按表格来

|          | 标头名称                     | 值                      |
| -------- | ---------------------------- | ----------------------- |
| 设置静态 | Access-Control-Allow-Methods | `GET,HEAD,POST,OPTIONS` |
| 设置静态 | Access-Control-Allow-Origin  | `*`                     |

保存即可
