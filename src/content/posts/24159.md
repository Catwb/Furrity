## 教程前置

1.[serv00账号](https://www.serv00.com/)  
2.ssh工具  
实在不行可以使用[webssh](https://ssh.090227.xyz/)  
3.一个少人用的梯子或者是梯子的地区

<!-- more -->

:::box{title="提醒 color:yellow"}
为什么要少人？serv00会**检测一个ip是否被大量用于注册**，此时如果你再使用这种ip或梯子，你会无法注册。（可能会提示**维护时间**等，在操作过程任何的无法注册都是因为ip问题）
:::



## 开始

来到页面，我们先点击这里

![](https://lc-gluttony.s3.amazonaws.com/IX0RrJ8UGC4d/wU6thJ0RzzyB14wz8P2OuyuziNa8R9gW/5BE82654967FCB600F990A65ADF2A416.png)

然后下滑，找到**“create account”**，并且点击。

来到如下页面：

![](https://lc-gluttony.s3.amazonaws.com/IX0RrJ8UGC4d/SbpbE0oqWSS6OGe2h5kufJ1DuLyw4RHt/2.png)

项目说明：

| 序号 | 内容                                             |
| ---- | ------------------------------------------------ |
| 1    | 乱填                                             |
| 2    | 乱填                                             |
| 3    | 用户名，决定了你的域名**（你有自己的域名除外）** |
| 4    | 邮箱地址                                         |
| 5    | 验证码                                           |
| 6    | `0` 或 `free`                                    |
| 7    | 勾选                                             |

当上方显示绿框则说明注册成功，需要去邮箱里面寻找邮件，邮件内容大概是这样

![](https://lc-gluttony.s3.amazonaws.com/IX0RrJ8UGC4d/jVAbiPza8zWHTLa162KxWr8UU5fqYN7f/P3.png)

接下来你就可以使用ssh来搭建你想要的网站（？）

不过要注意的是，使用serv00服务，需要你在退出面板、ssh的90天以内登录过panel或者是ssh，否则就会被销毁。不过你可以在github上找到自动保活源码并使用。

实际上serv00会不定时停止你的服务，你只需要**ssh一次或者登录面板一次**即可恢复服务。不出任何意外这个服务将会提供很久。
